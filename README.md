# 44ELITE - Penetration Testing Tool

44ELITE is a Python-based penetration testing tool that provides various functionalities for performing security assessments and testing.

## Features

- **Port Scanning**: Scan ports on a target machine to identify open ports.
- **Brute Force Login**: Perform brute force login attempts against a target with provided usernames and passwords.
- **Execute Commands**: Execute commands on a target machine.
- **Graphical User Interface (GUI)**: Provides an easy-to-use graphical interface for interacting with the tool.
- **Cross-Platform Compatibility**: Works on various operating systems that support Python.

## Requirements

- Python 3.x
- Tkinter (usually included with Python installations)

## Installation

1. Clone the repository:

git clone https://github.com/your_username/44ELITE.git


2. Navigate to the directory:

cd 44ELITE

3. Run the application:

python3 44ELITE.py


## Usage

1. Enter the target IP address and port range in the provided fields.
2. Use the buttons to perform actions such as scanning ports, brute force login, and executing commands.
3. View the output in the text area provided within the GUI.
